How to train

Dedupe is a deduplication library using fuzzy string matching with rules learned from active user input.

When selecting which columns to train on, ask yourself the following questions:

  • Can this column contain duplicate entries? If not, consider omitting it.
  • Does this field contain textual data or numeric data that represents an abstract concept? (e.g. zip codes) If not, consider omitting it.

Example fields that Dedupe may be appropriate for:

  • Names
  • Addresses
  • Concise Descriptions
